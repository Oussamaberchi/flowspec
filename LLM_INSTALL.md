# Install SpecFlow with LLM Assistance

If you prefer to have an AI Large Language Model (like ChatGPT, Claude, Gemini, etc.) guide you through the installation process, you can copy and paste the following prompt into the AI's chat interface.

## Installation Prompt

Copy the text below and paste it into your favorite LLM:

---

> "I want to install the SpecFlow CLI tool on my computer.
>
> Here is the GitHub repository URL: `https://github.com/Oussamaberchi/flowspec.git`
>
> Please generate a step-by-step installation guide for me based on this repository.
> The guide should include:
> 1.  **Cloning the repository**: Use `git clone https://github.com/Oussamaberchi/flowspec.git`.
> 2.  **Installing dependencies**: Explain how to use `npm install`.
> 3.  **Building the project**: Explain how to use `npm run build`.
> 4.  **Linking the command globally**: Explain how to use `npm link` so I can run `specflow` from anywhere.
> 5.  **Verifying the installation**: Show me how to run `specflow --version` or `specflow --help` to confirm it works.
>
> My operating system is [INSERT YOUR OS HERE (e.g., Windows, macOS, Linux)]. Please provide the exact terminal commands I need to run."

---

## Troubleshooting

If you encounter any errors during the process, copy the error message from your terminal and paste it back into the LLM chat. The AI will help you debug the issue.
